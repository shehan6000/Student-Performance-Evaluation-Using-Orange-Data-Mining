This project analyzes and evaluates student performance using the **Orange Data Mining** software platform. The goal is to explore student-related factors, visualize patterns, and build predictive models to understand academic performance.

The project demonstrates skills in:

* Data preprocessing
* Data visualization
* Machine learning
* Educational data analysis
* Workflow design using Orange

## Tools & Technologies

* Orange Data Mining
* Python (optional for dataset preparation)
* CSV Dataset
* Machine Learning Algorithms

## Dataset

The project uses a **Student Performance Dataset** containing information such as:

* Student demographics
* Study time
* Attendance
* Previous grades
* Family background
* Internet access
* Extracurricular activities
* Final exam performance

Example target variable:

* `Final Grade`
* `Pass/Fail`
* `Performance Category`


---

# Objectives

The main objectives of this project are:

1. Analyze factors affecting student performance
2. Identify relationships between study habits and grades
3. Predict student outcomes using machine learning
4. Visualize academic trends
5. Build an easy-to-understand Orange workflow

---

# Orange Workflow

The workflow used in Orange includes the following widgets:

1. **File** – Load dataset
2. **Data Table** – View records
3. **Select Columns** – Choose important features
4. **Impute** – Handle missing values
5. **Distributions** – Analyze feature distributions
6. **Box Plot** – Compare student groups
7. **Scatter Plot** – Visualize relationships
8. **Test & Score** – Evaluate models
9. **Random Forest**
10. **Logistic Regression**
11. **Naive Bayes**
12. **Confusion Matrix**
